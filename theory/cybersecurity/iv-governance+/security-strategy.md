# Security strategy

Steps to build security strategy

1. Identify the target state: define business requirements and objectives (ideal state).
2. Evaluate the current state: Assess the current security posture
3. Conduct gap analysis: identify the gaps between the current and desired states.
4. Develop the strategy: create a plan to remediate the gaps and align security with business objectives.
5. Create a security program: implement specific pojects and plans to enhance security controls.
